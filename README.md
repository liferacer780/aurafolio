# Multi-Chain Wallet Tracker

Premium portfolio tracker for multi-chain crypto assets with real-time analytics and transaction monitoring.

## Framework

Static HTML with Vanilla JavaScript, Chart.js for data visualization

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Features

- **Animated portfolio summary** with real-time value counter and 24h P&L
- **Multi-chain filtering** (Ethereum, Solana, BSC)
- **Donut chart allocation** with interactive legend
- **Token holdings table** with live prices and 24h changes
- **Transaction history** with send/receive visualization
- **Glassmorphic design** with backdrop-blur and depth
- **Responsive layout** optimized for desktop and mobile
